1. Define database in your own words

Database is an organised collection of data where data is stored electronically so it can be easily accessed , managed and updated

2. Difference between data & database

Data

Raw facts and values,Unorganized, Can be stored anywhere

Database

Organized collection of related data ,Well-organized (tables, rows, columns), Stored systematically in DBMS

3. What is DBMS

DBMS stands for Database Management System

A DBMS is a software system that alllows to store, retreive, update and manages data in a database

4. What is SQL used for?

SQL is used to communicate with a database to store, retrieve, update, and manage data.

5. Explain row & column

In a table, data is stored in the form of rows and columns.

Row
  • A row represents one complete record in a table.
  • Each row contains data for all columns.
Column
  • A column represents a single attribute or field.
  • Each column stores the same type of data.
